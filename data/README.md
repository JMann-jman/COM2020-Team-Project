This folder contains synthetic datasets for development.

Noise Seed Dataset v1 (Synthetic)
=================================
Purpose
- Seed data for COMM2020 "Neighbourhood Noise Explorer and Mitigation Planner" Team Project.
- Fully synthetic, zone-level, anonymised (no personal data).

Licence
- Created by the project team for coursework use.

Files
- zones.csv
- zone_adjacency.csv
- noise_observations.csv
- incident_reports.csv
- moderation_decisions.csv
- interventions.csv
- missions.csv
- scenarios.csv
- scenario_plans.csv

Timestamp format
- ISO 8601 UTC with Z suffix.

Notes
- Includes a few injected "event spike" intervals in event-tagged zones to support hotspot validation tests.
